Why this dashboard?

Data from the official WIRES-X active node list is now surfaced in a modern, responsive dashboard for quick insights.

  • Compact summary cards
  • Fast navigation between map and table views
  • Dark/light mode for all screens

Use this home page to understand node coverage, geolocation availability, and how fresh the data is without scrolling through raw tables.

About this project

This dashboard is built on the Yaesu WIRES-X active node list to improve usability and make the data easier to explore.

  • Aggregates active Wires-X nodes from the official Yaesu source.
  • Offers map and searchable table views for quick inspection.
  • Includes a JSON endpoint for integration with other tools.

Data is refreshed approximately every 20 minutes, matching Yaesu's update cadence, and is cached to reduce repeated load on the source.

Geolocation pins reflect the coordinates provided by room owners and may not always be exact.

Navigation

  • Home — overview and project context.
  • Maps — interactive node map with location pins.
  • Nodes — searchable table of all active rooms.
  • JSON — raw data feed for developers.

Built with

  • Flask
  • Bootstrap
  • Leaflet.js
  • OpenStreetMap
